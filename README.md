# Ansel

![](ansel.jpg)

## Prerequistes

As of today, only **Ubuntu 14.04 & 16.04 LTS** 64-bits are supported.

In order to be able to run, ansel requires the following libraries:

```bash
sudo apt-get install zlib1g-dev libudev-dev
```

## Build from sources

```bash
sudo npm install -g gulp
npm install
gulp
```
